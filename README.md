# Film_Compressor_Local
On Anaconda, you can follow :
•	Create a new virtual environment with Python 3.13 called for instance "Film_compress'.  please : 

o	install miniconda : https://www.anaconda.com/docs/getting-started/miniconda/install#linux-terminal-installer . Normally there is no need for sudo so you should be able to do it yourself. Then, close and reopen your terminal. If you want and if you know what you are doing, you can use any other package manager (micromamba, venv,...) 

o	once it's done, create the environment with the following command in the terminal 
conda create -n Film_compress python=3.13

•	Enter the virtual environment (with miniconda or your favorite package manager):
conda activate Film_compress

•	Then, install the necessary packages:
pip install moviepy

You also need ffmpeg, check online
